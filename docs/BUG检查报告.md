### 🔍 正在检查的潜在BUG

#### 后端相关
1. **接口权限验证**：检查所有接口是否都有正确的权限验证，防止越权访问
2. **参数校验**：检查所有接口的入参校验是否完善，防止非法参数导致报错
3. **SQL注入风险**：检查SQL查询是否使用了参数化查询，防止注入风险
4. **错误处理**：检查异常捕获是否完善，避免500错误直接暴露给前端
5. **数据库约束**：检查所有表的字段约束是否合理，避免空值和数据不一致
6. **密码安全性**：检查密码加密方式是否安全，存储是否符合安全规范
7. **JWT Token有效期**：检查Token过期时间是否合理，刷新机制是否完善

#### 前端相关
1. **页面响应式**：检查各页面在移动端的显示是否正常
2. **表单验证**：检查所有表单是否有前端验证，避免非法参数提交
3. **加载状态**：检查接口请求时是否有加载状态，防止重复提交
4. **异常处理**：检查接口请求失败时是否有友好的错误提示
5. **路由守卫**：检查需要登录的页面是否有路由守卫，防止未登录访问
6. **状态管理**：检查用户状态、登录状态的管理是否正确
7. **浏览器兼容性**：检查在不同浏览器下的显示是否正常

#### 功能逻辑相关
1. **习题提交判分**：检查代码题、选择题的判分逻辑是否正确
2. **学习进度统计**：检查学习进度、完成率的计算是否正确
3. **权限控制**：检查普通用户和管理员的权限边界是否清晰
4. **数据一致性**：检查关联数据的增删改是否保持一致性
5. **分页逻辑**：检查所有列表的分页逻辑是否正确
6. **文件上传**：检查文件上传的大小、格式限制是否完善，防止恶意文件上传
